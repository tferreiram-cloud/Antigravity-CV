Solutions Architect Microsoft - TD SYNNEX

Responsável por impulsionar a adição de novos clientes de GitHub Copilot no Azure, em parceria com ecossistemas indiretos.  

  

**Responsabilidades e Atribuições**  

* Aproveitar os dados do Azure do cliente para orientar a execução de go\-to\-market (GTM) na com o GitHub Copilot/Innovate with AI Apps and Agents.
* Influenciar a liderança para estabelecer a comunidade de revendas, fornecendo feedback e melhores práticas à equipe do GitHub.
* Ativar revendas e acelerar as atividades de GTM e vendas de GitHub Copilot para impulsionar a aquisição de novos clientes (novos "logos") de GitHub Copilot.
* Participação obrigatória nas atividades e eventos do programa SureStep da Microsoft e do GitHub.
* Foco em construir uma comunidade de revendas com 25\-50 parceiros dispostos a desenvolver uma prática de GitHub Copilot. Realizar abordagem direcionada aos parceiros para obter o comprometimento das revendas (gives/gets) em desenvolver a prática de GitHub nas frentes de implementação, marketing, produto e vendas, a fim de gerar novas conversas com clientes.
* Desenvolver programas e ofertas da Microsoft (campanhas) no Azure. Capacitar parceiros e realizar demonstrações para fomentar novas conversas com clientes.
* Elaborar e executar um plano com cada parceiro para atingir metas mensais de aquisição de novos clientes.
* Implementar um ritmo de negócios (RoB) com vendedores prioritários, GitHub e stakeholders da Microsoft (PDM, PSA, Channel Sales) para impulsionar o progresso.

  

**Requisitos e Qualificações**  

* Formação completa em Gestão de TI, Redes, Ciência da Computação, Sistemas ou áreas relacionadas.
* Experiência em empresa de serviços de TI, parceiro de Azure ou fornecedor de tecnologia com competências reconhecidas.
* Compreensão básica de Azure, aplicativos, desenvolvimento, IA e do ecossistema de parceiros da Microsoft.
* Inglês fluente
* Certificação Azure Fundamentals AZ\-900, AZ\-200, AZ\-204;
* Desejável conhecimento em GitHub Foundations
